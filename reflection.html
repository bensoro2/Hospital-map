สิ่งที่ได้เรียนรู้จากโปรเจกต์นี้

  • ความเข้าใจในเทคโนโลยีเว็บพื้นฐาน (HTML, CSS, JavaScript) และการนำมาประยุกต์ใช้สร้างเว็บแอปพลิเคชัน
  • การใช้งานไลบรารี Leaflet.js สำหรับการแสดงผลแผนที่ออนไลน์, การเพิ่มหมุด (Markers), และการจัดการ Popup
  • ความเข้าใจเกี่ยวกับรูปแบบข้อมูล GeoJSON และการนำมาใช้เพื่อแสดงข้อมูลเชิงพื้นที่พร้อมรายละเอียดประกอบ
  • การจัดการโครงสร้างโปรเจกต์เว็บเบื้องต้น โดยแบ่งแยกไฟล์ตามหน้าที่ (HTML, CSS, JS)
  • ทักษะการแก้ปัญหาที่เกิดขึ้นระหว่างการพัฒนา เช่น ปัญหาการโหลดข้อมูล (CORS) หรือการแสดงผล

แนวคิดในการต่อยอด

  • เพิ่มระบบค้นหา (Search) หรือกรอง (Filter) ข้อมูลโรงพยาบาลตามชื่อ หรือตามพื้นที่
  • เพิ่มข้อมูลจุดสนใจประเภทอื่นๆ เช่น สวนสาธารณะ, สถานีขนส่ง พร้อมไอคอนที่แตกต่างกัน
  • ปรับปรุงหน้าตา (UI) และประสบการณ์ใช้งาน (UX) ให้สวยงามและใช้งานง่ายขึ้น อาจใช้ CSS Framework ช่วย
  • เพิ่มความสามารถในการแสดงเส้นทางไปยังจุดที่เลือก หรือแสดงจุดที่อยู่ใกล้ตำแหน่งปัจจุบันของผู้ใช้
  • พัฒนาให้ผู้ใช้สามารถเพิ่มข้อมูลจุดสนใจของตัวเองลงบนแผนที่ได้ (อาจต้องมีระบบ Backend เพิ่มเติม)
  • เชื่อมต่อกับ API ภายนอกเพื่อดึงข้อมูลที่เป็นปัจจุบันมากขึ้น
