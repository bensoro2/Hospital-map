หัวข้อที่ 1: HTML - โครงสร้างเนื้อหาเว็บ

HTML (HyperText Markup Language) เป็นภาษาหลักในการสร้างโครงสร้างและกำหนดเนื้อหาต่างๆ บนหน้าเว็บไซต์ เราได้เรียนรู้เกี่ยวกับแท็ก (Tag) พื้นฐานต่างๆ เช่น <h1> ถึง <h6> สำหรับหัวข้อ, <p> สำหรับย่อหน้า, <a> สำหรับลิงก์, <img> สำหรับรูปภาพ, <ul>, <ol>, <li> สำหรับรายการ, และแท็กเชิงความหมาย (Semantic Tags) เช่น <header>, <nav>, <main>, <section>, <footer> เพื่อช่วยจัดระเบียบโค้ดและเพิ่มความเข้าใจให้กับทั้งผู้พัฒนาและ Search Engines รวมถึงการใช้ Attributes ต่างๆ เพื่อกำหนดคุณสมบัติเพิ่มเติมให้กับ Elements.

หัวข้อที่ 2: CSS - การจัดรูปแบบและตกแต่ง

CSS (Cascading Style Sheets) ใช้สำหรับควบคุมการแสดงผลและตกแต่งหน้าตาของ HTML Elements ให้สวยงามและน่าใช้งาน เราได้เรียนรู้วิธีการเลือก (Select) Elements ที่ต้องการจัดรูปแบบโดยใช้ Selectors ต่างๆ (เช่น Tag, Class, ID) และการกำหนดคุณสมบัติ (Properties) เช่น color, background-color, font-size, margin, padding, border. รวมถึงการทำความเข้าใจเกี่ยวกับ Box Model, การจัดวางองค์ประกอบ (Layout) ด้วย Flexbox หรือ Grid, และการทำให้เว็บไซต์รองรับการแสดงผลบนอุปกรณ์ต่างๆ (Responsive Design) โดยใช้ Media Queries.

หัวข้อที่ 3: JavaScript - การเพิ่มการโต้ตอบและพลวัต

JavaScript เป็นภาษาโปรแกรมที่ทำให้เว็บไซต์มีชีวิตชีวาและสามารถโต้ตอบกับผู้ใช้ได้ เราได้เรียนรู้พื้นฐานไวยากรณ์ (Syntax) ของ JavaScript, ตัวแปร (Variables), ชนิดข้อมูล (Data Types), เงื่อนไข (Conditions), ลูป (Loops), และฟังก์ชัน (Functions). ที่สำคัญคือการเรียนรู้วิธีการเข้าถึงและจัดการองค์ประกอบ HTML (DOM Manipulation) เพื่อเปลี่ยนแปลงเนื้อหาหรือสไตล์แบบไดนามิก, การจัดการกับเหตุการณ์ของผู้ใช้ (Event Handling) เช่น การคลิก การเลื่อนเมาส์ หรือการกรอกฟอร์ม, และการทำงานแบบ Asynchronous เช่น การดึงข้อมูลจากไฟล์ (เหมือนที่ใช้กับ GeoJSON) หรือ API ภายนอก เพื่อสร้างประสบการณ์การใช้งานที่ดีและซับซ้อนยิ่งขึ้น.
